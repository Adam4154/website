This system is controlled remotely via Adafruit IO website for IoT integration, using WiFi and MQTT protocol.

Project Introduction

A comprehensive overview of our Smart Room Control System

1

What is the Smart Room Control System?

Our Smart Room Control System is an Internet of Things (IoT) solution that automates various room functions including lighting that will depend to the light intensity and window operation (OPENING & CLOSING) via LDR sensor, climate control throughout Digital humidity and temperature sensor (DHT11), , and ultrasonic security monitoring that detect an intruder that come closer the door . The system uses sensors to monitor environmental conditions and automatically adjusts room settings for optimal comfort and energy efficiency.

2

Key Features

  • Automatic Lighting: Lights turn on/off based on ambient light levels or light intensity
  • Climate Control: Fan activates when temperature exceeds settle thresholds
  • Window Automation: Windows open/close based on light intensity
  • Ultrasonic Security: Motion detection with buzzer and LED alerts when objects are 30cm or further
  • Remote Monitoring: Real-time control and monitoring via Adafruit IO dashboard
3

Technology Stack

The system is built using ESP32 microcontroller, various sensors (DHT11, LDR, Ultrasonic), actuators (relays, servo motors, fan and lamp), indicators (LEDs and buzzer), display system(LCD_I2C Screen) and communicates via MQTT protocol with Adafruit IO for remote access and control.

Working Principle & Technical Explanation

How our Smart Room Control System operates

1

Sensor Data Collection

The system continuously monitors environmental conditions using DHT11 (temperature/humidity), LDR (light), and HC-SR04 ultrasonic sensors(detecting for movementof substance). This data is processed by the ESP32 microcontroller(as the brain of the system) which serves as the central processing unit, receiving, processing, and sending commands to control actuators within whole system.

2

Automated Decision Making

Based on predefined thresholds, the system automatically controls devices:

  • Lights turn ON when LDR value < 950, OFF when > 1050
  • Fan activates when temperature exceeds 28°C
  • Windows open when light level is sufficient and close when dark
  • Buzzer activates for 2 seconds when objects are detected at 30cm or further
3

Ultrasonic Security System

The HC-SR04 ultrasonic sensor continuously measures distance. When objects are detected at 30cm or further, the system triggers a 2-second buzzer alarm and activates the red LED. Green LED indicates safe distance (objects closer than 30cm).

4

MQTT Communication

The system connects to Adafruit IO via MQTT protocol to publish sensor data and receive control commands through the dashboard, enabling remote monitoring and control from anywhere.

5

Manual Override

All automated functions can be manually controlled remotely via MQTT commands to Adafruit IO.

Project Components

Hardware and software components used in our Smart Home Automation System

ESP32 Microcontroller

The brain of our system, handling all sensor data processing, system control, and WiFi communication through the dashboard.

DHT11 Sensor

Digital humidity and temperature sensor that measures environmental conditions for automatic climate control and fan operation.

LDR Light Sensor

Light Dependent Resistor that detects ambient light levels for automatic lighting control based on light intensity.

HC-SR04 Ultrasonic Sensor

Distance measurement sensor for security monitoring, triggers alerts when objects are detected at 30cm or further.

Relay Modules

Controls high-power devices like lights and fans safely with low-voltage signals from the ESP32.

Servo Motor

Controls window opening and closing based on light levels and automation rules with 180-degree movement.

12V DC Fan

Cooling fan controlled by relay module, automatically turns on when temperature exceeds 28°C for climate control.

Lamp

Lighting device controlled by relay module, automatically turns on/off based on ambient light levels detected by LDR sensor.

LED Indicators

Green and red LEDs used to indicate security status - green for safe distance, red for object detection.

Buzzer

Audio alert device that sounds for 2 seconds when objects are detected at 30cm or further for security alerts.

// Ultrasonic sensor distance measurement
float measureDistance() {
    digitalWrite(TRIG_PIN, LOW);
    delayMicroseconds(2);
    digitalWrite(TRIG_PIN, HIGH);
    delayMicroseconds(10);
    digitalWrite(TRIG_PIN, LOW);
    
    long duration = pulseIn(ECHO_PIN, HIGH);
    float distance = duration * 0.034 / 2;
    
    return distance;
}

// Ultrasonic security system handling
void handleUltrasonicSensor() {
    float distance = measureDistance();
    
    // Check if object is 30 cm or further away (>= 30cm)
    if (distance >= ULTRASONIC_DISTANCE_THRESHOLD && distance > 0) {
        if (!objectFarDetected) {
            // Object detected at 30cm or further - activate buzzer and red LED
            objectFarDetected = true;
            digitalWrite(BUZZER_PIN, HIGH);
            digitalWrite(RED_LED, HIGH);
            digitalWrite(GREEN_LED, LOW);
            buzzerActive = true;
            lastBuzzerTime = millis();
            
            Serial.print("🚨 BUZZER ON! Object at: ");
            Serial.print(distance);
            Serial.println(" cm (>= 30cm)");
        }
    } else {
        // Object is closer than 30cm (< 30cm)
        if (objectFarDetected) {
            objectFarDetected = false;
            Serial.println("✅ Object is close (< 30cm) - No alert");
        }
        
        // Green LED ON when object is close (< 30cm)
        digitalWrite(GREEN_LED, HIGH);
        digitalWrite(RED_LED, LOW);
    }
    
    // Turn off buzzer after 2 seconds
    if (buzzerActive && (millis() - lastBuzzerTime >= buzzerDuration)) {
        digitalWrite(BUZZER_PIN, LOW);
        buzzerActive = false;
        Serial.println("🔇 Buzzer turned off after 2 seconds");
    }
}

// Automatic lighting control based on LDR sensor
void automaticControl() {
    int ldrValues = analogRead(LDR_PIN);
    float temp = readTemperature();
    
    // Light and window auto control
    if (!manualLightControl && !manualWindowControl) {
        if (ldrValues < LIGHT_ON_THRESHOLD) {
            digitalWrite(RELAY1_PIN, HIGH);
            windowServo.write(90);
        } else if (ldrValues > LIGHT_OFF_THRESHOLD) {
            digitalWrite(RELAY1_PIN, LOW);
            windowServo.write(0);
        }
    }
    
    // Fan auto control
    if (!manualFanControl && temp > -100 && temp >= TEMP_FAN_ON) {
        digitalWrite(RELAY2_PIN, HIGH);
    } else if (!manualFanControl) {
        digitalWrite(RELAY2_PIN, LOW);
    }
}
